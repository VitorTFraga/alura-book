# AluraBooks 📚

Este é um projeto de uma landing page para uma livraria online, desenvolvido durante os estudos de Front-End na **Alura**. O objetivo principal foi aplicar conceitos práticos de design responsivo e arquitetura de CSS.

---

## 💻 Sobre o Projeto

O **AluraBooks** foi construído focando na experiência do usuário em diversos dispositivos. O desenvolvimento seguiu a metodologia **Mobile First**, onde a interface foi pensada primeiramente para dispositivos móveis e, posteriormente, adaptada para tablets e desktops através de Media Queries.

### 🛠 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica.
* **CSS3**: Estilização com foco em:
    * **Flexbox & Grid**: Para layouts dinâmicos.
    * **Media Queries**: Para responsividade.
    * **Variáveis CSS**: Para uma manutenção mais limpa.
* **SwiperJS**: Biblioteca de terceiros para criação de carrosséis interativos.

---

## 📱 Responsividade e Mobile First

Abaixo, os principais pontos trabalhados:
* **Menu Hambúrguer**: Implementado para otimizar espaço em telas pequenas.
* **Carrosséis Dinâmicos**: Navegação suave entre os livros em destaque.
* **Layout Adaptável**: Mudança de grid e posicionamento de elementos conforme o tamanho da tela (Mobile, Tablet e Desktop).
